# AI개발 수행내용

## 1. 사업과제

**Dubai 유가와 환율 등 거시경제 변수를 활용한 다음 달 소비자물가 상승률 예측 모델 개발 및 시각화**

## 2. 개요 및 현황

### 2.1 추진배경 및 목적

○ 국제유가의 변동은 원유 수입가격뿐 아니라 에너지비, 운송비 및 생산비의 변화를 통해 국내 소비자물가에 영향을 줄 수 있다. 이러한 영향은 즉시 반영되기보다 일정한 시차를 두고 나타날 가능성이 있으므로, 현재 유가와 과거 유가 변동을 함께 고려한 분석이 필요하다.

○ 국내 원유 수입 여건과의 연관성을 고려하여 국제유가 변수로 Dubai 유가를 선정하였다. 월별 Dubai 유가의 현재 변동률과 1개월, 2개월, 3개월 및 6개월 시차 변수를 구성하여 유가 변화가 이후 물가 예측에 제공하는 정보를 확인하고자 하였다.

○ 소비자물가는 유가만으로 결정되지 않으므로 원/달러 환율, 한국은행 기준금리 및 현재 소비자물가 상승률을 함께 활용하였다. 이를 통해 수입물가 여건, 통화정책 환경 및 물가의 자기 지속성을 종합적으로 반영하고자 하였다.

○ 한국은행 경제통계시스템(ECOS)의 월별 거시경제 데이터를 수집·정제하여 다음 달 소비자물가 전년 동월 대비 상승률을 예측하는 머신러닝 모델을 개발하고, 기준 모델과 여러 회귀 모델의 성능을 비교하여 최종 모델을 선정하는 것을 목적으로 한다.

○ 최종 예측 결과와 영향 요인을 사용자가 쉽게 확인할 수 있도록 Streamlit 기반 대시보드를 구현하였다. 사용자가 유가, 환율, 기준금리 등의 조건을 변경하여 시나리오별 전망 차이를 확인하고, GPT 기반 해석 기능을 통해 모델 결과를 자연어로 이해할 수 있도록 구성하였다.

### 2.2 과제 범위

| 과제 구분 | 내용 |
|---|---|
| 데이터 수집 | 한국은행 ECOS의 소비자물가지수, Dubai 유가, 원/달러 환율 및 한국은행 기준금리 월별 데이터 수집 |
| 데이터셋 구축 | 2005년 1월부터 2026년 5월까지의 자료를 월 단위로 변환·통합하고 분석용 데이터셋 구축 |
| 데이터 전처리 | 날짜 형식 통일, 가로형 원자료의 세로형 변환, 수치형 변환, 월별 집계, 결측치 확인 및 변수 간 기준월 정렬 |
| 탐색적 데이터 분석 | 주요 변수의 시계열 추이, 기술통계, 상관관계 히트맵 및 유가 시차별 상관관계 분석 |
| 변수 생성 | CPI 전년 동월 대비 상승률, 유가·환율 월간 변동률, 유가 1·2·3·6개월 시차 및 다음 달 CPI 예측 대상 변수 생성 |
| 모델 개발 | Persistence Baseline, Linear Regression, Ridge Regression, Random Forest 및 XGBoost 모델 학습·비교 |
| 성능 평가 | 시간 순서를 유지한 학습·테스트 분리와 MAE, RMSE, R² 평가지표를 이용한 모델 성능 평가 |
| 효과 검증 | CPI 단독 모델, CPI·환율·금리 모델 및 유가 변수를 포함한 전체 모델의 성능 비교를 통한 유가 정보의 추가적 예측효과 확인 |
| 서비스 구현 | Streamlit 기반 다음 달 CPI 전망, 시나리오 비교, 요인별 예측 기여도 및 데이터·모델 정보 화면 구현 |
| 결과 해석 | 규칙 기반 한 줄 해석과 GPT 기반 그래프 해석 기능을 통한 예측 결과 설명 |
| 테스트 | 데이터 로딩, 모델 추론, 시나리오 입력 변경, 그래프 표시 및 GPT 응답 기능 테스트 |

### 2.3 과제 추진 방법

#### 1) 구축 대상 선정 기준

○ **데이터 접근성 및 신뢰성**

- 한국은행 ECOS에서 제공하는 공개 통계를 활용할 수 있어 자료의 출처와 갱신 기준이 명확한지 검토하였다.
- 분석에 필요한 소비자물가, 국제유가, 환율 및 기준금리 자료를 공통된 월 단위로 수집·관리할 수 있는지 확인하였다.
- 2005년 이후의 장기 시계열을 확보하여 학습과 테스트에 필요한 관측치가 충분한지 검토하였다.

○ **예측모델 개발 적합성**

- 다음 달 소비자물가 상승률이라는 종속변수가 명확하고 정량적으로 측정 가능한지 확인하였다.
- 현재 물가 흐름과 유가, 환율, 기준금리 등 경제적으로 설명 가능한 독립변수를 구성할 수 있는지 검토하였다.
- 유가의 영향이 시차를 두고 반영될 가능성을 분석할 수 있도록 과거 유가 변수를 생성할 수 있는지 확인하였다.
- 시계열 데이터의 시간 순서를 유지하면서 기준 모델과 머신러닝 모델을 공정하게 비교할 수 있는지 검토하였다.

○ **활용성 및 확장 가능성**

- 다음 달 물가 방향과 예상 변화폭을 제시하여 경제지표 이해를 위한 참고자료로 활용할 수 있는지 검토하였다.
- 유가·환율·금리 조건을 변경하는 시나리오 분석을 통해 거시경제 변수 변화에 따른 전망 차이를 확인할 수 있도록 하였다.
- 향후 원자재 가격, 수입물가지수, 생산자물가지수 등 추가 변수를 결합하여 모델을 확장할 수 있는 구조인지 고려하였다.

#### 2) AI 예측 분석모델 적용 대상

|---|---|
| 적용 기능 | 다음 달 소비자물가 전년 동월 대비 상승률 예측 |
| 수집 데이터 | 소비자물가지수, Dubai 유가, 원/달러 환율, 한국은행 기준금리 |
| 주요 독립변수 | 현재 CPI 상승률, Dubai 유가 월간 변동률, 유가 1·2·3·6개월 시차, 원/달러 환율 월간 변동률, 기준금리 |

| 활용 화면 | 시계열 추이, 실제값·예측값 비교, 영향 요인, 시나리오 비교, 데이터·모델 정보 및 GPT 해석 |

#### 3) AI 분석모델 구축 프로세스

| 단계 | 주요 수행내용 |
|---|---|
| DATA IMPORTING | 한국은행 ECOS 원자료 수집, 월별 자료 구조 확인, 날짜 및 수치 형식 정리 |
| DATA PREPROCESSING | 가로형 자료의 세로형 변환, 월별 집계, 데이터 통합, 결측치 확인 및 시차 변수 생성 |
| USER INSIGHT | 주요 변수 추이 시각화, 상관관계 분석, 유가 시차별 관계 확인 및 예측 변수 선정 |
| MODELING | 시간 순서 기반 학습·테스트 분리, Persistence Baseline 구축, 회귀 모델 학습 및 성능 비교 |
| AUTO PREDICTION | 최종 선형회귀 모델을 이용한 다음 달 CPI 상승률 예측 및 모델 저장 |
| VISUALIZATION | Streamlit 기반 전망 카드, 추이 그래프, 영향 요인, 시나리오 비교 및 모델 정보 시각화 |
| INTERPRETATION | 규칙 기반 전망 방향 안내와 GPT 기반 그래프·예측 결과 해석 제공 |

> 본 모델은 거시경제 변수와 소비자물가 사이의 통계적 예측 관계를 활용한다. 개별 변수의 변화가 소비자물가 변화를 직접 유발한다는 인과관계를 의미하지 않으며, 예측 결과는 경제적 의사결정을 위한 단독 근거가 아닌 참고자료로 활용해야 한다.
